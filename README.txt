
📦 تطبيق تحليل المكونات الغذائية من صورة باستخدام Streamlit + GPT-4 + OCR

🔧 المتطلبات:
- Python 3.8+
- تثبيت Tesseract OCR:
  - Windows: https://github.com/tesseract-ocr/tesseract
  - Linux/Mac: sudo apt install tesseract-ocr

📥 التثبيت:
1. pip install -r requirements.txt
2. ضع مفتاح OpenAI في متغير بيئة OPENAI_API_KEY أو داخل الكود مباشرة

🚀 التشغيل:
streamlit run app.py
